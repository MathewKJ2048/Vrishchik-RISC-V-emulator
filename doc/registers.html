Registers

Standard RISC-V provides 32 registers for use by the programmer. Each register has a size of 4 bytes. A summary of their uses is listed.

number name description saved by
x0 $zero always zero
x1 $ra return address caller
x2 $sp stack pointer callee
x3 $gp global pointer
x4 $tp thread pointer
x5 $t0 temporary return address caller
x6-7 $t1-2 temporary caller
x8 $s0 / $fp saved register / frame pointer callee
x9 $s1 saved register callee
x10-11 $a0-1 function argument / return value caller
x12-17 $a2-7 function argument caller
x18-27 $s2-11 saved register callee
x28-31 $t3-6 temporary caller
